FAQ

How do I add a post?

  1. Export the article from Typora as HTML.
  2. Create a folder under posts/, for example posts/my-note/.
  3. Put the exported article into posts/my-note/index.html.
  4. Add a card to index.html, a row to archives.html, and a search item to search.html.

What should a post page contain?

Use posts/first-post/index.html as the template. Keep the shared header, footer, stylesheet, and script links, then replace the title, metadata, and article body.

Do I need a build command?

No. This site is plain static HTML, CSS, and JavaScript. A local static server is enough for previewing.
