This is a sample article page for the static HTML version of Fungus Field.

When you export a post from Typora, keep this page shell and replace the content inside <div class="post-content typora-content"> with the exported article body. Edit the title, date, URL, summary, and tags only in posts.json; the generator rewrites the post header and list pages from that file.

Suggested workflow

  1. Create a folder like posts/my-new-post/.
  2. Copy this file to posts/my-new-post/index.html.
  3. Replace only the article body inside <div class="post-content typora-content">.
  4. Add or update title, date, url, summary, and tags in posts.json.
  5. Run python3 scripts/generate-list-pages.py.
