This is a sample article page for the static HTML version of Fungus Field.

When you export a post from Typora, keep this page shell and replace the content inside <div class="post-content"> with the exported article body. If Typora exports a complete HTML document, copy only the article body into this region so the site keeps the same navigation, theme toggle, and typography.

Suggested workflow

  1. Create a folder like posts/my-new-post/.
  2. Copy this file to posts/my-new-post/index.html.
  3. Replace the title, date, tags, meta description, and article body.
  4. Add links on the home, archive, and search pages.
